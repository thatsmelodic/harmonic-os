# The Harmonic Codex

The Harmonic Codex is the source of truth for Harmonic OS.

Every developer, AI agent, and future collaborator must use this folder before building features.

## Volumes

1. `01-masterplan.md` — product vision, phases, platform overview.
2. `02-ai-rulebook.md` — rules for Copilot and all AI coding agents.
3. `03-lore.md` — story, symbolism, Cup, Heart, frequencies, worlds.
4. `04-brand-dna.md` — visual identity, color, typography, motion, UI language.
5. `05-public-os.md` — visitor-facing Harmonic OS experience.
6. `06-control-hub.md` — admin/control system for editing site content, colors, seasons, and worlds.
7. `07-season-engine.md` — seasonal/drop-based theme system.
8. `08-investor-mode.md` — polished investor/sponsor/collaborator presentation mode.
9. `09-analytics.md` — tracking, dashboards, conversion data, platform intelligence.
10. `10-transitions.md` — animation/transition rules.
11. `11-content-system.md` — central content model for music, videos, drops, socials, announcements.
12. `12-roadmap.md` — build order and release phases.

## North Star

Harmonic OS must never feel like a basic portfolio, template, or static link page.

It should feel like a visitor is booting into Melodic's universe: premium, intentional, cinematic, mobile-first, and alive.
