# Investor Mode

## Purpose

Investor Mode is a polished presentation experience for investors, sponsors, collaborators, manufacturers, agencies, and business partners.

It should feel like a cinematic pitch deck inside Harmonic OS.

## Audience

Investor Mode is for:

- Investors.
- Sponsors.
- Collaborators.
- Agencies.
- Brand partners.
- Manufacturers.
- Mentors.
- Grant reviewers.

## Required Sections

- Founder story.
- What Harmonic OS is.
- Brand ecosystem.
- Four Worlds.
- Audience and community.
- Revenue streams.
- Traction and proof.
- Product roadmap.
- Partnership opportunities.
- Contact CTA.

## Revenue Streams To Track

- Clothing sales.
- Music streams/content.
- YouTube/Facebook monetization.
- Patreon/community subscriptions.
- Brand sponsorships.
- Events/pop-ups.
- Collaborations.
- Future software/platform features.

## Design Direction

Investor Mode should be cleaner and more business-focused than the main OS, while still using Harmonic visual DNA.

Use:

- Premium black background.
- Controlled purple glow.
- Sharp charts/cards.
- Founder narrative.
- Strong numbers.
- Clear CTA.

## Build Strategy

Do not build full Investor Mode in v0.1.

First, document the structure. Later, create a route like `/investor` or an unlockable mode from the OS.
