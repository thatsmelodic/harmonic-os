# Creator Control Hub

## Purpose

The Control Hub is the private owner system for editing Harmonic OS without needing to edit code.

It should feel like the internal cockpit of the OS.

Possible names:

- Harmony Core
- Harmony Engine
- Pulse
- Control Hub

Working name: **Harmony Core**.

## Core Editing Powers

The owner should eventually be able to edit:

- Site colors.
- Active season/theme.
- Homepage hero text.
- Featured world.
- Featured music.
- Featured videos.
- Featured clothing drops.
- Featured food/basketball episodes.
- Social links.
- Announcements.
- Countdown timers.
- World descriptions.
- Investor Mode data.

## Control Hub Sections

### Dashboard

Quick overview of what is live, what is featured, and current platform status.

### Theme Editor

Controls global look:

- Primary color.
- Glow color.
- Accent color.
- Background effect.
- Motion intensity.
- Season.

### Content Manager

Controls content across worlds:

- Songs.
- Music videos.
- YouTube videos.
- Clothing drops.
- Basketball episodes.
- Food reviews.
- Announcements.

### Social Manager

Controls all social links in one place.

### Season Manager

Activates Fall, Winter, Spring, Summer, and special drop themes.

### Investor Manager

Updates investor presentation content.

## Build Strategy

Do not build the full Control Hub in v0.1.

Start by designing public site content as structured data. Later, connect that data to editable admin forms.
